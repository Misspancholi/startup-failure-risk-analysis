Startup Failure Risk Analysis & Prediction
- Problem Statement :

Analyze startup funding and survival patterns to identify key predictors of business failure and build a predictive risk model.

- Dataset :

66K+ startup records
Cleaned to 41K after preprocessing
Features include funding, rounds, survival duration

- Key Insights :

Startups with fewer than 3 funding rounds show nearly 2x higher failure risk.

Funding continuity is a stronger predictor than total funding amount.

Failure recall improved from 12% to 87% after feature enhancement.

-Model Performance

Accuracy: 94%
Failure Recall: 87%
